# ☕ Espresso-Emporium

**Espresso-Emporium** is a premium, real-time web application for coffee enthusiasts. Experience the country's finest coffee culture with our modern marketplace, interactive dashboards, and real-time community features.

---

## 🚀 Key Features

### 🛒 Marketplace & Community
- **Curated Selection:** Browse and discover premium coffee products from top sellers nationwide.
- **Real-time Chat:** Connect instantly with sellers and other coffee lovers via our integrated messaging system.
- **Ratings & Reviews:** Share your experiences and browse community feedback with detailed product reviews.
- **Advanced Filtering:** Easily find what you need with robust category-based filtering.

### 💼 Professional POS System
- **Dynamic Interaction:** High-performance POS interface with intuitive grab-and-drag scrolling.
- **Audio Feedback:** Integrated sound effects for a responsive and engaging checkout experience.
- **Order Management:** Streamlined workflow for managing sales and customer interactions.

### 🔐 Role-Based Access Control
- **Buyers:** Personalized browsing, simplified ordering, and chat access.
- **Sellers:** Comprehensive product lifecycle management (Add, Update, Delete) and buyer engagement.
- **Admins:** Full-spectrum platform oversight including user management, content moderation, and activity monitoring.

---

## 🛠️ Technologies Used

### Frontend
- **React 19:** Building a fast, interactive user interface.
- **Vite:** Next-generation frontend tooling for a lightning-fast development experience.
- **Tailwind CSS 4:** Crafting a stunning, modern design with the latest CSS utility framework.
- **React Router 7:** Seamless navigation and deep linking.
- **Lucide-React & React Icons:** Premium iconography.
- **Recharts:** Dynamic data visualization for dashboard insights.
- **SweetAlert2 & Lottie:** Engaging animations and interactive notifications.

### Backend & Real-time
- **Custom Node.js/Express Backend:** Powering complex business logic and data persistence.
- **MongoDB:** Scalable NoSQL database for flexible data management.
- **Socket.io:** Enabling seamless, real-time communication for chat and updates.
- **Firebase Auth:** Industry-standard secure authentication.

## 🔖 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sumyta-Bentey-Habib/Espresso-Emporium.git
   cd Espresso-Emporium
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file with your environment variables (Firebase, API URLs, etc.).

4. **Run Development Server:**
   ```bash
   npm run dev
   ```
